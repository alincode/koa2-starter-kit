# Koa2 Starter Kit

```
npm start
npm run dev
npm run dev-pm2
npm run lint
npm run watch
```

### env

* node 8.6
* npm 5.3
* mysql 5.6

### pm2

```
pm2 start bin/ecosystem_staging.json
```
